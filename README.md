# Notebooks

A collection of applied data science notebooks covering forecasting, causality,
synthetic data, and generative AI evaluation.

## Notebook Index

| Notebook | Description |
| --- | --- |
| [Forecasting: Calendar and Event Features](notebooks/forecasting_calendar_event_features.ipynb) | Crash course on building causally informed calendar and event features for forecasting. Uses a synthetic daily sales dataset to handle missing dates, encode holidays and other events, fit a simple model, and interpret feature impact. |
| [Mapping Causality: Socioeconomic Status and Health](notebooks/mapping_causality_socioeconomic_status_health.ipynb) | Introduces causal thinking for health data by examining how socioeconomic status can shape health outcomes. Covers causal framing, simulated SES-health data, missing values, categorical encoding, and naive versus causally informed modeling. |
| [The Rise of Synthetic Data](notebooks/synthetic_data_when_real_data_isnt_enough.ipynb) | Demonstrates synthetic tabular data generation for privacy-sensitive domains. Builds a healthcare-style dataset, trains a CTGAN model, and compares real and synthetic data for fidelity, utility, and privacy-oriented use cases. |
| [Worked Example: Website Traffic Forecasting](notebooks/worked_example_website_traffic_forecasting.ipynb) | Applies calendar and event feature engineering to daily website traffic. Highlights campaign-driven traffic spikes and shows how weekend and campaign indicators help explain forecast patterns. |
| [Worked Examples in Generative AI Prompt Engineering and Evaluation](notebooks/generative_ai_prompt_engineering_evaluation_examples.ipynb) | Compares prompt engineering and evaluation patterns across geography question answering and arithmetic word problems. Shows how dataset type changes the usefulness of prompting strategies and simple metrics. |

## Usage

Open the notebooks in Jupyter, VS Code, or GitHub's notebook viewer. Each
notebook is self-contained and includes its own walkthrough, code examples, and
discussion sections.
